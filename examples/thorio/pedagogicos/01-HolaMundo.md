# Ejemplo 01 - Hola mundo

Este es el ejemplo mas pequeno posible en Thorio.

La idea pedagogica es mostrar que todo programa:

- inicia con `inicio`
- termina con `fin`
- puede escribir texto con `mostrar`

```thorio
inicio
  mostrar "Hola mundo"
fin
```

## Que aprende el lector

- estructura minima de un programa
- salida por pantalla
