# ThorioLang

Documentacion publica para aprender a programar con Thorio.

Este repositorio esta pensado para personas que quieren:

- entender que es Thorio
- aprender programacion paso a paso en espanol
- practicar con ejemplos pequenos
- consultar la sintaxis del lenguaje cuando ya avanzaron un poco

## Principios editoriales

- Todo el contenido se escribe en Markdown.
- Los diagramas usan Mermaid cuando ayudan a explicar mejor una idea.
- Los ejemplos usan notacion real de Thorio.
- Los ejemplos pueden inspirarse en el repositorio tecnico `Thorio`, pero aqui se adaptan para ensenanza.
- Cada archivo en `examples/` incluye frontmatter con `runtime`, `uses`, `ide_web_runnable` e `ide_web_reason`.
